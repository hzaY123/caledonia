## üìö Usage
After running Caledonia, you can use several commands to manage your calendar:

- **Add an Event:**
   ```
   caledonia add "Meeting with team" --date "2023-10-15" --time "14:00"
   ```
- **List Events:**
   ```
   caledonia list
   ```
- **Remove an Event:**
   ```
   caledonia remove "Meeting with team"
   ```
